STUDENT REGISTRATION SYSTEM

The Student Registration System allows users to register students and manage their information dynamically. Below are the key features of the system:

  • Form Validation: Ensures all fields are filled, validates email format, and checks for numeric values for Student ID and Contact No.
  • Toast Notifications: Displays success or error messages for various actions like adding, editing, or deleting students.
  • Dynamic Table: Lists all registered students and updates automatically when a new student is added or existing data is modified.
  • Edit and Delete Options: Allows users to edit or delete a student's information directly from the table.
  • Local Storage Persistence: Ensures that the student data remains even after the page is reloaded.
